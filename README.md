Supernova Roket Takımı - Yer Kontrol İstasyonu

Bu yazılım, TEKNOFEST Roket Yarışması teknik şartnamesine tam uyumlu olarak Python ve PyQt6 kütüphanesi kullanılarak geliştirilmiştir.
Teknik Özellikler:
  Veri Frekansı: Saniyede 5 paket (5 Hz) veri işleme kapasitesi.
  Haberleşme Kontrolü: UKB aktif edildikten sonra başlayan otomatik 2 dakikalık haberleşme sayacı.
  Fail-safe: Bağlantı koptuğunda son GPS verilerini dondurma ve veri yedekleme mekanizması.
  Hakem Ekranı: Teknofest şartnamesi madde 4.6.38 gereği tasarlanan canlı takip arayüzü.
